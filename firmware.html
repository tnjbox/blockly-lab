Blockly Lab 韌體燒錄中心

請依照手上的控制板版本選擇正確韌體。ESP8266 / ESP32 系列可使用 ESP Web Tools 線上燒錄；Arduino UNO 版本需另行使用 Arduino IDE 燒錄。

燒錄前請確認：
  1. 使用 Chrome 或 Edge 瀏覽器。
  2. USB 線必須支援資料傳輸。
  3. 請先關閉 Arduino IDE 序列埠監控視窗，避免 COM 埠被占用。
  4. 燒錄過程中不要拔除 USB 線。
ESP8266

D8 WS2812 + 8 按鈕版

適用於完整 SmartRingController：D8 接 12 顆 WS2812，並使用 8 個按鈕輸入。

韌體版本：C8

檔案：SmartRingController_D8_C8.bin

ESP8266

D4 LED-only WS2812 版

適用於只有 D4 接 12 顆 WS2812 燈環的版本，沒有 8 個按鈕，按鍵狀態固定為未按下。

韌體版本：C8 LED-only D4

檔案：Ws2812Ring_D4_C8.bin
